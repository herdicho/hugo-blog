== Local Machine Setup

. _Install_ Nodejs dan Yarn Package Manager
. _Clone_ dari Git
. Arahkan _terminal_ ke _directory Git_ dan jalankan perintah `yarn install`
. _Run App_ menggunakan perintah `yarn start`
